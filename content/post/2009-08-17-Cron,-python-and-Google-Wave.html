--- date: 2009-08-17T00:00:00Z title: Cron, python and Google Wave url: /2009/08/17/Cron,-python-and-Google-Wave/ --- Documentation on getting a google wave robot written in python working with the Cron functionality is still a little sparse.


Getting a cron job into the capabilities.xml file.

Add the following directive to the main function of your python program:

myRobot.RegisterCronJob('/path',60)
