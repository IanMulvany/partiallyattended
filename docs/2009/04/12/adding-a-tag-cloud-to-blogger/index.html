    Tags: tags, blogger, tag cloud, widget

That led me to the Phydeaux3 blog where there is a kick ass implementation of a widget to display a tag cloud for blogger. Hopefully you can see that it works pretty well. Neat plugin.

    Hopefully I'll be able to dissect this to put in place some of the ideas I have regarding journals.
